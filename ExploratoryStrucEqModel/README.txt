Scripts to conduct Exploratory Structural EQ modelling.

Method isnt very popular (yet) but basically it allows modelling of item cross loadings on other components then the item where the item mainly / mostly contributes to.
Best paper written so far is from Asparouhov, Muthén & Muthén; https://www.statmodel.com/download/EFACFA810.pdf
See statmodel.com for more information.
